*Project Overview*

This project demonstrates a modern architecture with focus on:

* MVVM + Clean Architecture
* Jetpack Compose
* Paging 3
* Coroutine and Kotlin Flow
* Repository pattern
* Mapper pattern
* DI (Using Koin)
* Testing strategies (Unit, Integration i UI testing)
